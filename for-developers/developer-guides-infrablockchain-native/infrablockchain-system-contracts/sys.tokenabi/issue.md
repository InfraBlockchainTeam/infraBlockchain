# issue

## Issue a token

To issue a new token call the issue`()` action.

#### Action Parameters

| Field    | Type   | Example       | Description              |
| -------- | ------ | ------------- | ------------------------ |
| to       | name   | "eosio"       | Account for issue token  |
| quantity | asset  | "1.0000 DUSD" | Token quantity for issue |
| memo     | string | "New issue"   | memo                     |
